# Excel Logic for Automation of Irregular Banking Data

# Objective 
Automate the classification of irregular banking trxs descriptions using word-based logic;

# Problem found

Banking tsx data is inconsistent and unstructured text , making manual classification slow;

# Solution 
Used modular Excel logic using LAMBDA and text parsing functions to identify and classify tsxs automatically;

## Example Logic 
Fee detection (transfer charges, commissions, financial taxes);
Payroll-related transactions;
Interest-related entries;
Payment providers (e.g., Stripe).

# Functions used
LET/LAMBDA,text functions;

# Outcome
Accurate classification based on the formulas;
Consistency across 3 collumns;
Scalable to Large Datasets reducing load time.
